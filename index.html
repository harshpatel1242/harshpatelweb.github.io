Appraisal Management System(Web-MySQL)

A system was developed to allow employees to request appraisals, which are reviewed by higher authorities based on the employee's work history. The system uses a common database with cross-validation credentials and the Google Auth API for login and sign-up. Access to the portal is granted on a per-employee basis, with different levels of accessibility. The system also includes a machine learning model to detect an employee's overall performance score and display it using the Charts.js library, which is intended to assist higher authorities in decision-making. The system was implemented using a combination of HTML, CSS, JavaScript, Jquery, Bootstrap, PHP, MySQL, Python, and the scikit-learn library.
